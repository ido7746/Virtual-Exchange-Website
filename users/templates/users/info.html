Virtual Market Web

This website allows users to manage virtual stock portfolio using real time market data. Users determine their cash anount and they can buy and sell stocks from several word wide markets. This project writen in python using Django library and pulls market information from tradingviews API. User identities managed in a sqlite3 database with custom credential for each user.
